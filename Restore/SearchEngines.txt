## 一、关键字搜索



早些时候，浏览器是有专门的“**搜索栏**”的。



然而现在大部分浏览器，都已经将“**搜索栏**”与“**地址栏**”合并，取而代之的方法，是通过“**关键字**”来调用**搜索引擎**。





### 1、使用方法



在使用“**关键字搜索**”之前，我们需要先添加搜索引擎。



包括三个部分：**名称、关键字、搜索链接**。



名称和关键字可以随便写，而“**搜索链接**”的获得则需要一些观察。



我们以 B站为例，在网站的搜索框里输入任意内容，比如 aaa，点击“**搜索**”，然后观察地址栏链接的变化。



![S03_关键字搜索_Firefox](_Images/S03 - 快捷搜索/S03_关键字搜索_Firefox.png)



复制该条链接，然后将链接中的 **aaa** 换成 **%s**，即可获得“**搜索链接**”。



<img src="_Images/S03 - 快捷搜索/S03_关键字搜索_Firefox2.png" alt="S03_关键字搜索_Firefox2"  />



所以，现在浏览器的搜索流程是：**关键字 -> 空格 -> 搜索内容。**







### 2、使用心得



添加搜索引擎倒不是很难，难主难在如何记住这些搜索引擎的“**关键字**”。



我的心得是：



**1、常用搜索引擎，使用单个字母作为关键字。**



比如百度、谷歌、必应、搜狗的关键字分别为：b、g、y、s。



百度：b

谷歌：g

必应：y

搜狗：s



**2、类似搜索引擎，使用字母组合作为关键字。**



比如百度图片、谷歌图片、必应图片、搜狗图片的关键字分别为：bi、gi、yi、si。



百度图片：bi

谷歌图片：gi

必应图片：yi

搜狗图片：si（i 是 images 的意思）



又比如是同一网站的不同搜索内容，比如百度图片、百度百科、百度知道、百度网盘的关键字，可以这样设置：



百度：b

百度图片：bi（Baidu Images）

百度百科：bk（Baidu Baike）bb 已被占用

百度翻译：bf（Baidu Fanyi）

百度文库：bw（Baidu Wenku）

百度知道：bz（Baidu Zhidao）

百度网盘：bp（Baidu Pan）



谷歌：g

谷歌美国：gg（设置为 gg，主要是为了快速搜索）

谷歌好手气：gl（Google Lucky）

谷歌图片：gi（Google Images）

谷歌视频：gv（Google Videos）

谷歌翻译：gt（Google Translate）



搜狗：s

搜狗图片：si（Sogou Images）

搜狗微信：sw（Sougou Weixin）





**3、不常用搜索引擎，使用字母组合作为关键字（不超过 3 个）。**



比如 GitHub、GreasyFork、PixaBay、DouBan、YouKu，分别取两个字母作为关键字。



DouBan 豆瓣：db

YouKu 优酷：yk

GitHub：gh

GreasyFork：gf

PixaBay：pb



而如果某些搜索引擎的关键字为 3 个字，那么也可以简化为 2 个字母。



比如 Chrome、Edge、Firefox 的扩展商店搜索，原定设置的关键字是 crx、edg、add，但为了输入方便，我们可以将关键字改为：crr、edd、add。



Chrome Web Store：crr

Microsoft Edge Store：edd

Firefox Addons：add

## 三、多重搜索



那如果我想要同时搜索多个引擎呢？



比如我们想同时搜索"**百度**"和"**谷歌**"，来对比不同的搜索结果，这时就需要用到“**多重搜索**”了。



这个功能可以通过扩展“**Custom Search Engine**”来实现。（支持 [Firefox](https://addons.mozilla.org/zh-CN/firefox/addon/custom-search-engine/) / [Chrome](https://chrome.google.com/webstore/detail/custom-search-engine/kelahdmegihhooaelnaahkeggodajdjf)）







### 1、使用方法



首先，在地址栏输入 **ms** 启用多重搜索。



然后，就如同浏览器自带的“**关键字搜索**”一样，先输入“**关键字**”，然后**按一下“空格”，**再输入“**搜索内容**”。



浏览器自带搜索流程：                        **关键字 -> 空格 -> 搜索内容。**

多重搜索的搜索流程：**ms -> 空格 -> 关键字 -> 空格 -> 搜索内容。**



**使用方法：**



- 搜索某个引擎：ms 关键字1 搜索内容
- 搜索多个引擎：ms 关键字1,关键字2 搜索内容
- 搜索某个引擎分组：ms @分组1 搜索内容
- 搜索多个引擎分组：ms @分组1,分组2 搜索内容
- 搜索所有引擎：ms @ 搜索内容



**英文解释：**



- 搜索某个引擎：ms key1 TEXT
- 搜索多个引擎：ms key1,key2 TEXT
- 搜索某个引擎分组：ms @cat1 TEXT
- 搜索多个引擎分组：ms @cat1,cat2 TEXT
- 搜索所有引擎：ms @ TEXT



**注意事项：**



- ms 是 Multi Search 的缩写，快捷键无法修改。
- key 关键字不能用大写，否则无法调用。
- 搜索链接里的 %s 要用 {searchTerms} 来代替。
- 不用在意排序的问题，点击表格的第一行，表格会自动排列。





### 2、使用心得



说实话，这个扩展上手还是有点难度的，可一旦你知道了它的设计逻辑，你就会发现它的强大。



首先，我们可以将扩展的“**Key关键字**”，与“**浏览器自身的搜索关键字**”保持一致，这样可以省去记忆成本。



其次，“**Category分组**”可用可不用，但如果用了，那么我们还是可以用“**浏览器自身的搜索关键字**”来命名。



比如，百度、谷歌、必应的“**Key关键字**”分别是 **b、g、y**。



我们可以将它们分为一组，组成了“**网页搜索三连**”，然后，我们将“**Category分组**”的命名设置为 **b**。



也就是说，



每一个“**Category 分组**”，谁是“**老大**”，那么分组的命名就用谁的“**Key 关键字**”，这样也是为了节省记忆成本。



同样，比如“**视频搜索三连**”的分组，里面有“**Youtube、B 站、谷歌视频**”。



Youtube 是老大，Key关键字是 **yt**，那么 **Category 分组**也命名为 **yt**，使用 **ms @yt** 就可以同时搜索这三个引擎。



不用担心它命名会重复，因为要调用这个“**Category分组**”，需要在前面加上 **@** 才行。



所以，通篇下来，我们只需要记住浏览器本身的那套“**关键字**”就行了。



此外，同一个搜索引擎可以分到多个“**Category分组**”。



也就是说，这个“**Category分组**”其实是“**标签**”，同一个搜索引擎可以同时打上多个"**标签**"。比如：



网页搜索三连：b

百度、谷歌、必应



谷歌搜索三连：ggg

谷歌、谷歌图片、谷歌视频



所以，“**谷歌**”这个搜索引擎的“**Category分组**”有两个：b 和 ggg。

## 四、拖拽搜索



搜索的另一个情况，就是文字本身就是在网页上。



比如平时上网浏览时，遇到一些不太懂的词，最简单的搜索方法，就是选中该词，然后**向右拖拽来搜索**。





### 1、使用方法



大多数浏览器本身并没有**拖拽搜索**功能，我们可以通过扩展来实现。



Chrome / Edge 浏览器推荐扩展 **crxMouse Gestures**，Firefox 则推荐 **GlitterDrag**。我的使用方法是：



向右拖拽：**百度搜索**

向上拖拽：**谷歌搜索**

向左拖拽：翻译文字

向下拖拽：复制文字

## 六、全局搜索



上面讲的内容，都是在浏览器内进行的操作，那如果是浏览器外呢？





### 1、Windows 全局搜索



**MouseInc** 是一款小巧的鼠标手势软件，功能丰富，且资源占用非常低。



它有一个“**复制增强功能**”，可以在任意窗口选中文字后，连续按两下 **Ctrl+C** ，那么会弹出一个快捷操作菜单，然后就可以调用系统默认浏览器来搜索了。



![S03_Windows全局搜索](_Images/S03 - 快捷搜索/S03_Windows全局搜索.png)



这里有一个小技巧：



将菜单第一个选项的“**百度搜索**”，快捷键设置为 **C**，那么选中文字后，按两下 **Ctrl+C** 后，再按一下 **C** 键，就可以一键搜索，省去选择菜单的时间。



关于 **MouseInc** 的使用方法，我在文章《**i08 - 鼠标手势**》中已经做过详细介绍。



此外，你还可以回复相应的关键字查看下面的文章。比如：



i08《鼠标手势》：一键复制搜索

i10《翻译工具》：一键复制翻译

C14《文件搜索》：系统全局搜索





### 2、Mac 全局搜索



Mac 系统自带的“**聚焦搜索**”功能就可以全局搜索。



按 **Cmd+空格** 呼出搜索窗口，输入要搜索的内容，再按一下 **Cmd+B** 就可以调用系统默认浏览器来搜索了。
